Javascript modules

a brief introduction

Why do we need modules?

  • expose functionality and hide implementation
  • segregate our code
  • scoping

How about JS?

1. inline script

	<html>
	  <head>
		<!-- ... -->
	  </head>
	  <body>
		<!-- ... -->
		<script>
		  
		</script>
	  </body>
	</html>
  • - global pollution
  • - no code segregation
  • - no encapsulation
1. file dep + inline script

             <html>
            <head>
              <!-- ... -->
            </head>
            <body>
              <!-- ... -->
              <script src="lib.js"></script>
              <script>
            	
              </script>
            </body>
             </html>
  • + code segregation
  • - global pollution
  • - no encapsulation
